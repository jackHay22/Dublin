Configuration

Game settings and configuration can be found in src/dublin/config.clj.

  • Environment defines a set of mapsets and the current mapset.
  • Mapset defines the set of map layers associated with the set, the tileset associated with the mapset, and any objects (animated, functionality-driven tiles).
  • Layer defines a layer a mapset and contains the 2D map array, the map x and y positions, the map width and height, and the start draw x and y values.
  • Tileset defines a set of map images (sheet) and the tile dimension.
  • MapObject defines images associated with object (sheet), the object dimension, action marker, control key, current frame, and the operating? condition.
